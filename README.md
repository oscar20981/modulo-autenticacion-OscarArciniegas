Objetivo del módulo



Este módulo tiene como propósito crear el sistema de autenticación para la aplicación del proyecto MEJ.

Incluye funciones como registro, inicio de sesión, cierre de sesión y manejo de roles.



Historias de usuario:

* H1- Login de usuarios
* H2- Registro de nuevos usuarios.
* H3- Visualización de un dashboard al iniciar sesión
